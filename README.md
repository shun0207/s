# Build Assist Client 26.1.2

Build workspace for the Minecraft Fabric 26.1.2 client-only Build Assist mod.

The release JAR is compiled against Java 25, Fabric Loader 0.19.3, and Fabric API 0.153.0+26.1.2.

Build Assist 2.0 adds an Effortless Building-style radial mode menu, line/floor/wall/cuboid/hollow-box/circle/cylinder/sphere/stairs modes, and optional Litematica 0.27.5 automatic nearby placement using the block held in the main hand.
